# Blinkit Sales Data Analysis (SQL Project)

This project analyzes Blinkit retail sales data using SQL to identify key business insights related to product performance, outlet types and sales distribution.

Key Analysis Performed:
- Total sales and Average sales
- Average Ratings
- Sales contribution by outlet size and location
- Product category performance
- Fat content analysis 
- Outlet type sales comparison

SQL techniques used:
- Aggregations (SUM, AVG)
- Window functions
- CTEs
- Group By analysis
- Percentage contribution calculations

Key Insights:
1. Low Fat items had higher sales compared to regular Fat
2. Fruits, Vegetables and Snacks contributerd higher revenue whereas and Seafood contributed the lowest
3. Highest Revenue was in the year 2000 and 2010
4. Medium and Small outlets contributed highest percentage sales of 37.8% and 37.2% respectively
5. Tier 2 outlet contributed highest revenue

Tools Used:
SQL, Excel

Goal:

To demonstrate data analysis skills using SQL and derive actionable retail insights.
